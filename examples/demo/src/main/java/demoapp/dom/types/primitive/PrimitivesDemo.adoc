(since 1.x)

The framework supports following types in their primitive and boxed form:

* java.lang.*Boolean*
* java.lang.*Byte*
* java.lang.*Short*
* java.lang.*Integer*
* java.lang.*Long*
* java.lang.*Float*
* java.lang.*Double*

Also note how *null* and *void* are handled regarding action results or properties.
